Today, I started Hash Tables (or built-in dictionaries in Python).

```python
phonebook = {
    "Alice": "98765-43210",
    "Bob": "87654-32109"
}

phonebook["Charlie"] = "76543-21098"
bob_number = phonebook["Bob"]

print(bob_number)
print(phonebook)
```

Output:

```
87654-32109
{'Alice': '98765-43210', 'Bob': '87654-32109', 'Charlie': '76543-21098'}
```
